   The following information was adapted from Seumas Macdonald's [Greek Grammar terms in Greek](https://thepatrologist.com/2020/07/21/a-cheat-sheet-for-greek-grammar-terms/).

__Parts of Speech:__\
ὄνομα = noun\
ῥῆμα = verb\
ἐπίθετον = adjective\
ἐπίρρημα = adverb\
ἄρθρον = article\
προθέσις = preposition\
σύνδεσμος = conjunction\
ἐπίφθεγμα = interjection\
μόριον = particle\
μετοχή = participle\
ἀντωνυμία = pronoun

__Case (πτῶσις):__\
ὀνομαστική = nominative\
αἰτιατική = accusative\
γενική = genitive\
δοτική = dative\
κλητική = vocative

__Gender (γένος):__\
θηλυκόν = feminine\
ἀρσενικόν = masculine\
οὐδέτερον = neuter\
κοινόν = common

__Number (ἀριθμός):__\
ἑνικός = singular\
πληθυντικός = plural\
δυϊκός = dual

__Tense (χρόνος):__\
ἐνεστώς = present\
μέλλων = future\
ἀόριστος = aorist\
παρατακτικός = imperfect\
παρακείμενος = perfect\
ὑπερσυντελικός = pluperfect

__Voice (διάθεσις):__\
ἐνεργετική = active\
μέση = middle\
παθητική = passive

__Mood (ἔγκλισις):__\
ὁριστική = indicative\
προστακτική = imperative\
ὑποτακτική = subjunctive\
εὐκτική = optative\
ἀπαρέμφατος = infinitive

__Degree of Adjective:__\
θετικόν = positive\
συγκριτικόν = comparative\
ὑπερθετικόν = superlative
